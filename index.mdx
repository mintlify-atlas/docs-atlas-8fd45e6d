ProxyDroid is an open-source Android app that transparently redirects all network traffic on your device through a proxy server. It supports HTTP, SOCKS4, and SOCKS5 protocols, works with or without root access, and provides fine-grained control over which apps are proxied, which addresses are bypassed, and when the proxy connects automatically.

## Get started

<Steps>
  <Step title="Check requirements">
    ProxyDroid requires Android 5.0 (API 21) or later. For iptables mode, root access is also needed. See [Requirements](/requirements) for the full list.
  </Step>
  <Step title="Install the app">
    Build from source using Android Studio or Gradle and sideload the APK. See [Installation](/installation) for step-by-step instructions.
  </Step>
  <Step title="Configure your proxy">
    Open ProxyDroid, enter your proxy server's host and port, and select the proxy type. See [Proxy Settings](/configuration/proxy-settings).
  </Step>
  <Step title="Enable the proxy">
    Toggle the **Running** switch at the top of the app to start routing traffic through your proxy server.
  </Step>
</Steps>
